# Lab5 Art Space 应用实验报告

## 1. 应用内容
本应用展示3幅世界经典艺术作品：
1. 梵高《星空》
2. 达芬奇《蒙娜丽莎》
3. 蒙克《呐喊》

## 2. 界面结构说明
界面分为三大逻辑区块：
1. 作品展示墙：使用 Image + Surface 实现带边框、阴影的画框效果
2. 作品信息区：使用 Column + Text 显示作品标题、作者、年份
3. 控制按钮区：使用 Row + Button 放置 Previous / Next 切换按钮

布局嵌套：
- 主界面：Column（垂直排列所有区块）
- 按钮区：Row（水平放置两个按钮）
- 信息区：Column（垂直放置文字）

## 3. Compose 状态管理
使用 remember + mutableStateOf 定义状态变量 currentArtwork，初始值为1。
所有图片资源、文字信息都根据该状态变量动态获取，实现状态驱动UI。

## 4. 按钮条件逻辑
- Next 按钮：
  1→2，2→3，3→1（循环）
- Previous 按钮：
  3→2，2→1，1→3（循环）
均使用 when 表达式实现，逻辑清晰。

## 5. 遇到的问题与解决
1. 问题：图片没有画框效果
   解决：使用 Surface + border + elevation 实现画框与阴影
2. 问题：按钮点击后界面不更新
   解决：确保状态变量使用 by 委托 + mutableStateOf
3. 问题：边界切换错误
   解决：使用 when 处理边界值，实现循环切换
